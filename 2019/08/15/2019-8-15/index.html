Hexo博客搭建和部署

August997's Blog
2019-08-15

1.

配置git环境和node.js

git的安装与配置：https://www.cnblogs.com/yszr/p/8317147.html

node的安装与配置：https://www.runoob.com/nodejs/nodejs-install-setup.html

其中最重要的一步就是node的npm安装，记得要添加到全局环境（npm可以理解为node的工具包的安装器）

2.

接下里就是hexo的安装，打开命令控制台，输入

npm install -g hexo-cli

推荐使用淘宝镜像cnpm：

1.npm config set registry https://registry.npm.taobao.org

2.cnpm install -g hexo-cli

3.

完成后输入

hexo init <folder>

新建hexo目录文件夹，然后跳转到对应的文件夹、

cd <folder>

最后为此项目安装相关的依赖

npm install

到此为止，一个新的hexo项目已经创建完成。

4.

我们可以启动该项目查看自己的网页，在控制台中输入

hexo server

跳出localhost:4000点击即可看见自己的网页。关于博客的撰写和相关的配置这里就不过多说明，可以去Hexo的官网了解。https://hexo.io/zh-cn/

5.

接下来就是hexo的部署。我的方式是将自己的代码上传到github上让github进行托管，也可以上传到码云（码云的速度要快，比较推荐）。这里讲的是在github托管的例子：

  1. 先创建一个新的github仓库。仓库的名字格式为

    用户名.github.io

    比如我的用户名为August997，则命名为August997.github.io

  2. 创建完后，将仓库的地址复制到hexo项目根目录下_config.yml文件里的deplyment下，没有就自己手打（千万别忘了：后加一个空格，否则无效）

    # Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: https://gitee.com/WJRepositorys/August997.github.io.git
  branch: master

  3. 接下来就是要把本地的代码上传到github中去。首先还需要安装一个插件。

    cnpm install --save hexo-deployer-git

    安装完成之后，输入

    hexo d

    此时已经开始上传本地代码。

  4. 上传完成之后我们再次访问刚才创建的仓库就可以看见代码，此时在浏览器中输入

    https://August997.github.io

    就可以看到网站已经部署到github中去了。
